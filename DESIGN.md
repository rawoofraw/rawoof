# "Money Heist" Tactical Aesthetic Design System

## Core Aesthetic
Covert operation, classified heist blueprints, and surveillance feeds. The interface should feel like a tactical dashboard used to plan a high-stakes mission.

## Layout & Structure
- **Containers:** Sharp, rigid containers resembling physical dossiers or vault schematics.
- **Effects:** Glitch effects on hover states. Subtle CRT scanlines for a surveillance camera feel.

## Typography
- **Headers:** Bold, stencil-style or heavy sans-serif fonts (resembling stamped classified documents).
- **Body & Metrics:** Strict monospace fonts for all body text, code snippets, and metrics.

## Color Palette
- **Background (Primary):** Pure Black (`#000000`)
- **Background (Secondary/Panels):** Dark Charcoal (`#121212`)
- **Accent Color:** Crimson Red / "Dali Jumpsuit" (`#D30000`)
- **Text Color:** Stark White (`#FFFFFF`)

## Components
- **Cards (Projects/Experience):** Tactical mission briefs and blueprints. Red accents, white text on dark charcoal backgrounds.
- **Headers/Stamps:** Use red borders or red text to simulate classified stamps.
